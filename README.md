# SLEAP Workshop

Padilla-Coreano's Lab workshop on how to use SLEAP for Animal Pose Estimation. 

Make sure to download this repo by running 

~~~
git clone https://github.com/padillacoreanolab/SLEAP_workshop
~~~

We will be using some files, specially the ones in the [examples](https://github.com/padillacoreanolab/SLEAP_workshop/tree/master/examples) folder. 

Also please make sure to download this [video](https://www.dropbox.com/s/8k2ibmkg7ba95ma/CAGE2_2.1v1.3_20220520.AVI?dl=0) by following the instructions in Dropbox. 

Additionally, please refer to the [Instructions](https://github.com/padillacoreanolab/SLEAP_workshop/blob/master/Instructions.md) on how to install SLEAP and the [Workshop's guide](https://github.com/padillacoreanolab/SLEAP_workshop/blob/master/Workshop.md).
